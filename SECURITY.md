# Security Policy

## Supported versions

| Version | Supported |
| --- | --- |
| Current `master` and the latest 3.x pre-release | Yes |
| All earlier versions | No |

## Reporting a vulnerability

Use [GitHub private vulnerability reporting](https://github.com/trungdev05/laravel-backup-panel/security/advisories/new). Do not open a public issue or include sensitive details in a pull request.

Include the affected version, a minimal reproduction, impact, and any proposed mitigation. The maintainer will coordinate the fix and disclosure through the private report.
